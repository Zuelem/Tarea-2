# Tarea-2 11/02/2026

¿Qué significa la extensión de un archivo? Un sufijo/Un código de tres o cuatro letras que aparece al final del nombre de archivo e indica el tipo de archivo.

¿Qué es Markdown? Es un una forma simple  de darle formato a un texto usando solo el teclado además es compatible con documentos técnicos. En markddown es donde van las instrucciones , licencia, etc

¿Para qué se usan archivos .MD? Md es una extensión de archivo de textos markdown, es un texto simple y funcional para escribir y describir contenido con formato.
